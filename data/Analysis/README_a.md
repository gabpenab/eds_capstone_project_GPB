# Analysis data
This folder contains intermediate outputs generated during exploratory and analytical steps.

## Notebook
A document compiling notes on the analytical process, including reasoning, intermediate results, and interpretations. This complements the R scripts and provides context for methodological decisions.

## Climate cleaning
Outputs from the climate data processing stage, including correlation plots and other diagnostics used for variable selection.

## Maps model
Maps and graphical outputs generated from model fitting and projections. These serve as inputs for the final visualizations presented in /outputs.

## Model
This folder contains the full set of model outputs generated during the modeling process (e.g., individual runs and ensemble). Due to file size, it is stored externally and can be accessed here:
